## 🚀 Como Executar os Projetos

### Pré-requisitos
```bash
# JDK 11 ou superior
java -version

# Maven
mvn -version

# Docker (para projetos com Kafka)
docker --version
```

### Executando um Projeto Spring Boot
```bash
cd nome-do-projeto
mvn spring-boot:run
```

### Executando Testes
```bash
mvn test
```
